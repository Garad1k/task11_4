# Линейная регрессия: сравнение с "глупой моделью" (RMSE)

В этом задании строится модель линейной регрессии для предсказания цены автомобиля (**MSRP**) по признакам:

- Year
- Engine HP
- Engine Cylinders
- highway MPG
- city mpg
- Popularity

Качество модели сравнивается с “глупой моделью”, которая всегда предсказывает одно и то же значение — среднее MSRP на тренировочной выборке.  
Метрика качества: **RMSE**.
